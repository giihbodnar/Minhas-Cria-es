HTML E CSS

*index.html pagina padrão.
*o .html diz que o arquivo é uma pagina na web
*para formatar o texto utilizados tags, que são usadas com <tag>
*Informar sempre a lingua dentro da tag do html. ex: <html lang="pt-br">
*Linguagem unicode para caracteres especiais

TAGS ESTRUTURAIS
<!DOCTYPE html> - informa que estamos utilizando a ultima versão do html
<html> - fecha no fim do arquivo
<meta charset="utf-8"> - escolha de dicionario, "utf-8"
<title> - muda o nome da pagina na web
<head> - colocar os comandos 
<body> - colocar as coisas para aparecerem na tela
